Chapitre Bonus – Exercices Web

1. Que désigne « Le Web » ?

Le web, c’est toutes les ressources reliées entre elles par des liens sur lesquels on clique pour aller d’une page à une autre.

2. Quelle est la condition essentielle pour accéder au web ?

Il faut que la machine soit connectée à Internet.

3. Où sont stockées les ressources du web ?

Sur des serveurs distants.

4. Échange client-serveur

Le client envoie une demande, le serveur répond avec la ressource demandée (page, image, vidéo...).

5. Protocoles d'échange

HTTP et HTTPS.

6. Qu’est-ce qu’un URL ?

Un URL indique le protocole, le nom de domaine et le chemin vers une ressource.

7. Moteur de recherche

Un outil qui propose des URL selon les mots-clés de l'utilisateur.

8. Robots Crawler / Spider

Ils explorent automatiquement les pages web pour les indexer.

9. Comment les moteurs de recherche choisissent les résultats ?

Ils comparent les mots-clés avec leur base de données et utilisent des algorithmes de pertinence.

10. Une page web, c’est quoi ?

Du texte, des images, des vidéos, des sons, du code HTML.

11. Langage HTML

HTML fonctionne avec un système de balises.

12. Le CSS

Le CSS sert à styliser les pages : couleurs, tailles, polices, etc.

13. Réutilisation des pages

Non automatique. Exemples de licences : Creative Commons.
