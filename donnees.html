Données structurées

Les données structurées sont des informations organisées selon un format précis pour être facilement lues et utilisées par les ordinateurs.

1. Qu’est-ce qu’une donnée structurée ?

    Des informations rangées dans un tableau ou une base de données Faciles à chercher, trier ou analyser Exemples : liste d’élèves avec nom, prénom, âge

2. Formats de données

    CSV : données séparées par des virgules JSON : utilisé dans les sites web XML : un autre format lisible par les machines

3. Pourquoi c’est utile ?

    Pour stocker des informations dans un site ou une application Pour partager des données entre machines Pour analyser ou faire des statistiques
