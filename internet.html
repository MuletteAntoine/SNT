Internet et Web

Internet est le réseau qui relie les ordinateurs du monde entier. Le Web est un service qui utilise Internet pour afficher des pages web.

1. Différence entre Internet et Web

    Internet = les câbles, les connexions Web = les sites comme YouTube, Wikipedia, etc.

2. Comment fonctionne un site ?

    On tape une URL (ex : www.google.fr) Un serveur envoie la page Le navigateur l’affiche

3. Langages du Web

    HTML : structure la page CSS : habille la page (couleurs, styles) JavaScript : rend la page interactive

4. Enjeux

    Accès à l’information Problèmes de sécurité (piratage, virus) Protection des données
