Programmation Python

Python est un langage de programmation facile à apprendre et utilisé dans de nombreux domaines.

1. Afficher un message

print("Bonjour le monde !")

2. Les variables

nom = "Alice"
age = 15

print("Nom :", nom)
print("Âge :", age)

3. Les types de données

    int : entier (ex : 5) float : nombre à virgule (ex : 3.14) str : texte (ex : "Bonjour") bool : vrai ou faux (True ou False)

4. Les conditions

age = 16

if age >= 18:
    print("Majeur")
else:
    print("Mineur")

5. Les boucles

5.1. Boucle for

for i in range(5):
    print(i)

5.2. Boucle while

i = 0
while i < 3:
    print(i)
    i += 1

6. Les fonctions

def saluer(nom):
    print("Bonjour", nom)
